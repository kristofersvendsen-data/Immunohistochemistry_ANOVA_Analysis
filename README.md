# Immunohistochemistry_ANOVA_Analysis
This is an analysis of data generated using Gemini to be similar to a series of experiments completed during my MSc in Neuroscience ([Svendsen, 2023](https://hdl.handle.net/1880/116587)) and were later published in the Journal of Pharmacology and Experimental Therapeutics ([Svendsen et al., 2025](https://pubmed.ncbi.nlm.nih.gov/39921943/)).

A detailed description of the experiment and analysis are provided in an R Markdown (.Rmd) file. The original file is provided in this repository and can be opened with RStudio; however, the output of this file can be viewed [here](https://kristofersvendsen-data.github.io/Immunohistochemistry_ANOVA_Analysis/).
